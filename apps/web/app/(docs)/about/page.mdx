# 关于

[洛谷仓库](https://luogu.store)（以下简称“本平台”）是 oi.zone 开源项目生态的一部分。本平台致力于对公开可访问的信息进行长期保存、结构化整理与检索展示，并在此基础上提供内容关联、相关内容推荐与个性化推荐等服务，以提升 OI 相关知识与讨论内容的可检索性、可引用性与可追溯性。

本平台并非洛谷官方服务，与洛谷及其运营方不存在隶属、代理或保证关系（除非另行以书面形式明确）。

## 服务概述

1. 内容归档与索引：对公开内容进行抓取、存储、去重、索引与快照展示。
2. 检索与聚合：提供全文检索、标签聚合、引用链路、相关主题归并等能力。
3. 推荐服务：基于内容相似性、关联关系以及用户在本平台的使用行为，提供相关内容推荐与个性化推荐（详见[《隐私政策》](/privacy)）。
4. 开源协作：本平台的部分组件以开源方式发布，并接受社区贡献（详见[《招贤纳才》](/join)与相关仓库说明）。
