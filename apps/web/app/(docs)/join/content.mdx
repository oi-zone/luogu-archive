# 招贤纳才

oi.zone 是面向 OI 与算法竞赛社区的公益开源项目生态。我们希望将为广大 OIers 提供更便利的体验。我们欢迎认同这一目标的贡献者加入。

## 我们的方向

对于本项目：

1. 归档与检索：抓取、存储、索引、去重、快照渲染、引用与追溯。
2. 内容关联：标签体系、引用链、主题聚类、跨页面关联。
3. 推荐与排序：相关内容推荐、个性化推荐、离线评估与线上迭代。
4. 平台工程：性能、稳定性、风控、反滥用、可观测性与部署自动化。
5. 合规与安全：隐私保护流程、去标识化能力、安全响应与供应链安全。

对于 oi.zone 生态的其他项目，我们欢迎任何为 OI 社区带来积极影响的想法与实践。

## 主要技术栈

本项目当前主要使用以下技术栈与工程工具：

* TypeScript
* Python
* C#
* Node.js
* .NET
* React
* Next.js
* Tailwind CSS
* Drizzle
* PostgreSQL
* Redis
* pnpm
